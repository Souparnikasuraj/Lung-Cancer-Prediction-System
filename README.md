# Lung Cancer Prediction System

## Description
A machine learning web application built using Django that predicts the likelihood of lung cancer based on user input.

## Features
- User Registration and Login
- Lung Cancer Prediction
- Prediction History
- Comparison of Results
- Responsive User Interface

## Technologies Used
- Python
- Django
- HTML
- CSS
- SQLite
- Pandas
- NumPy
- Scikit-learn

## Installation

```bash
git clone https://github.com/Souparnika3/Lung-Cancer-Prediction-System.git

cd Lung-Cancer-Prediction-System

pip install -r requirements.txt

python manage.py runserver
```
